# RM PRO v1
<img width="200" height="160" alt="Logo-blanco" src="https://github.com/user-attachments/assets/5631fb49-080c-4007-911c-87f7e54234d3" />

RM PRO v1 es un indicador desarrollado para facilitar la gestión de riesgo directamente desde el gráfico en TradingView.

<img width="933" height="635" alt="Captura de pantalla 2026-05-13 180629" src="https://github.com/user-attachments/assets/43edb62a-3afd-4937-9bc5-0a63d452fbb4" />

La idea principal es simplificar el cálculo y visualización de operaciones de forma rápida e intuitiva, permitiendo ajustar setups visualmente mientras el sistema recalcula automáticamente toda la información del trade.

Actualmente el proyecto se encuentra en fase de pruebas y feedback.

---

# Características

✅ Cálculo automático de riesgo
✅ Tamaño de posición dinámico
✅ Take Profit automático
✅ ROE estimado
✅ Margen requerido
✅ Precio de liquidación
✅ Detección automática LONG / SHORT
✅ Configuración de RR personalizada
✅ Soporte para múltiples trades
✅ Movimiento histórico mediante OFFSET o TIME

# LONG / SHORT Automático

📈 LONG
Cuando el Entry está por encima del Stop Loss.

📉 SHORT
Cuando el Entry está por debajo del Stop Loss.

No es necesario seleccionar manualmente el tipo de operación.

---

# Cómo funciona

Solo necesitas configurar:

* Entry
* Stop Loss

El indicador automáticamente calculará:

* Riesgo
* Volumen
* TP
* ROE
* Liquidación
* Margen requerido

Además, los niveles pueden moverse visualmente desde el gráfico y el sistema actualizará automáticamente toda la operación.

# Instalación

1. Abrir TradingView
2. Entrar a cualquier gráfico
3. Abrir Pine Editor
4. Pegar el contenido del archivo `.pine`
5. Presionar "Add to Chart"
6. Guardar el script
